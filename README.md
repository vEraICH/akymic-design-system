# Akymic Design System

Shared design system repository.

## Packages
- `@akymic/tokens`: semantic tokens and CSS variables (light + dark)
- `@akymic/ui`: (placeholder) promoted shared components

## Intended workflow
- Paper.design is your canvas.
- Promote stable tokens and primitives into this repo.
- Apps consume the DS but can override per-app via their override layer.
