# @akymic/ui

Reusable React component package for the Akymic Design System.

## Components

| Package | Components |
|---------|-----------|
| `checkbox` | `Checkbox`, `CheckboxField` |
| `radio` | `RadioGroup`, `Radio` |
| `switch` | `Switch`, `SwitchField` |
| `tabs` | `Tabs`, `TabsList`, `TabsTrigger`, `TabsContent` |
| `breadcrumb` | `Breadcrumb`, `BreadcrumbItem`, `BreadcrumbLink`, `BreadcrumbSeparator`, `BreadcrumbEllipsis`, `BreadcrumbList` |
| `tooltip` | `Tooltip` |
| `dropdown` | `DropdownMenu`, `DropdownItem`, `DropdownSeparator`, `DropdownLabel` |

## Usage

```tsx
import { Checkbox, RadioGroup, Radio, Switch, Tabs, TabsList, TabsTrigger, TabsContent } from "@akymic/ui";
```

All components consume tokens from `packages/tokens/tokens/tokens.css`. No hard-coded colors.

See `docs/components.md` for full API reference and accessibility notes.
